## Execution notes (the practical part)

### 1) Personal Color Analysis

Core prompt style: strict step-by-step constraints + drift lock.

Sample image:
- https://pbs.twimg.com/media/HGr0oKIWcAAbXbi.jpg

Use when:
1. beauty creators need repeatable client visuals,
2. stylist pages need “diagnostic” content,
3. you want utility, not just aesthetics.

### 2) Nike Surreal Motion

Core prompt style: motion-first character direction + typography integration.

Sample image:
- https://pbs.twimg.com/media/HGr4CYOaYAA56pd.jpg

Use when:
1. sports campaign concepts,
2. high-energy thumbnails,
3. pitch decks that need visual drama fast.
