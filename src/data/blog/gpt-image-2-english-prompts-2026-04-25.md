## 高复用模板精拆

### 1) Personal Color Analysis

```text
Create a high-end, editorial-style personal color analysis infographic using the provided portrait...
```

为什么火：分层约束非常完整，能把“自拍图”稳定转为“咨询级报告图”。

样图链接：
- https://pbs.twimg.com/media/HGr0oKIWcAAbXbi.jpg

### 2) Nike Surreal Athletic Poster

```text
A highly detailed digital artwork in a surreal, dynamic style inspired by premium athletic advertising for Nike...
```

为什么火：把运动广告语法写到了材质、动作、字体遮挡关系这一级。

样图链接：
- https://pbs.twimg.com/media/HGr4CYOaYAA56pd.jpg

### 3) 4x4 Nostalgic Couple Grid

```text
4x4 grid of candid nostalgic photos shot with iphone of a young couple at vacation...
```

为什么火：极短提示词但风格识别强，几乎零门槛复制。

样图链接：
- https://pbs.twimg.com/media/HGuEU_7bQAAklr6.jpg

### 4) Structured Mirror Selfie JSON

```text
{
  "prompt": {
    "subject": {...},
    "appearance": {...},
    "clothing": {...},
    "environment": {...},
    "lighting": {...},
    "camera": {...},
    "style": {...}
  }
}
```

为什么火：JSON 结构使可控性极高，适合稳定批量生产。

样图链接：
- https://pbs.twimg.com/media/HGrgxuZb0AA0XEm.jpg

```text
A high-fashion surrealist poster for Rolex... Aspect ratio: 3:4
```

为什么火：奢侈品海报里最关键的“空间、材质、文案权重”都被明确了。

```text

```

为什么火：变量简单，粉丝内容和人物专题都能快速套用。

样图链接：

### 7) Minimal Product Branding

```text
A minimalist product advertisement with a fried chicken bucket placed on a clean white podium...
```

为什么火：极简品牌广告提示词短小但商业完成度高。

样图链接：
- https://pbs.twimg.com/media/HGup1wYagAAZt64.jpg

## 场景建议

1. 个人服务类账号：优先 1。
2. 品牌创意与广告：优先 2、5、7。
3. 社媒情绪内容：优先 3、4。
4. 人物与IP专题：优先 6。

## 常见误区

1. 只复制名词，不复制结构约束。
2. 不加参考图和负提示，导致人物和文本漂移。
3. 输出尺寸与发布平台不匹配，成图虽好看但难发布。
