## 我筛出来最值得立刻复用的 4 套

### 1) 人像编辑模板（黑雾、姿态、构图）

原帖：[wanerfu](https://x.com/wanerfu/status/2046845192490418688)

核心可复用提示词骨架：
```text
9:16 竖向，编辑肖像，单人主体，柔和黑雾滤镜，轻微朦胧，柔和高光泛光。
室内极简背景，负空间构图，主体略偏心。
人物：年轻亚洲女性，自然肤质，简约妆容，头发自然蓬松略凌乱。
姿势：坐姿，一腿弯曲一腿放松，身体微倾，肩膀不对齐，头部轻微倾斜。
光线：柔和侧光，阴影细腻，整体低调安静。
```

图片样例：
- https://pbs.twimg.com/media/HGfaZAgaQAA6ahr.jpg

适合场景：头像升级、IP 形象图、封面人像。

### 2) 人物履历信息图模板（结构化海报）

原帖：[liu10102525](https://x.com/liu10102525/status/2046847829063417868)

核心可复用提示词骨架：
```text
按示例 layout，research {人物}，生成人物海报。
必须包含：
1) 姓名 + slogan + 金句 + 肖像
2) 出生/教育/关键节点
3) 职业履历时间轴
4) 核心理念（4条）
5) 代表作品（6个）
6) 金句与演讲（3条）
7) 管理风格关键词（4个）
8) 未来方向（4个）
9) 底部总结 + 金句横幅
真实资料优先，虚构部分必须标注。
```

图片样例：
- https://pbs.twimg.com/media/HGfcHuVb0AAhR4U.jpg
- https://pbs.twimg.com/media/HGfcUDaaoAAmdsU.jpg

适合场景：个人品牌介绍、人物图鉴、公众号配图。

### 3) 电商直播场景模板（截图 + 口播动作）

原帖：[web3annie](https://x.com/web3annie/status/2046872809423712522)

核心可复用提示词骨架：
```text
生成 9:16 高清手机直播截图，电商直播间，黑金科技背景。
主播正对镜头，表情自信，情绪有感染力。
动作分三段：开场点镜头 -> 中段下压与前推动作 -> 结尾 1-2-3 倒数并指向下方链接。
加入实时弹幕、礼物特效、火箭升空光效。
口播：想上车的现在就上，来，123，上链接！
画质：商业广告级，面部细节清晰，光影层次分明。
```

图片样例：
- https://pbs.twimg.com/media/HGftKsGaAAAZBvb.jpg

适合场景：带货脚本可视化、短视频封面、A/B 素材测试。

### 4) 小红书科普切片模板（食物一生）

原帖：[GeekCatX](https://x.com/GeekCatX/status/2046986329347838441)

核心可复用提示词骨架：
```text
主题：{食物名称}的一生。
必须是同一个完整食物，横向切片展示时间变化：未成熟 -> 成熟 -> 过熟 -> 变质。
主体结构连续，不是多个食物拼接。
竖版 3:4，顶部标题“{食物名称}的一生”，每段含阶段名、外观、内部变化、食用建议。
风格：米色极简背景，真实摄影感，中文科普排版，适当使用警示/勾选标识。
```

图片样例：
- https://pbs.twimg.com/media/HGhZkoKb0AAkDsV.jpg

适合场景：小红书、公众号、品牌科普图。

## 中文区写法的共同规律

1. 结构化约束比“艺术辞藻”更重要。
2. 明确排版规则（对齐、字体、比例）能显著减少翻车。
3. 带占位符的模板（如 `{人物}`、`{食物名称}`）最适合批量复用。
4. 商业场景驱动很强，大家不只要好看，更要能变现。

## 可直接抄走的 2 条优化建议

1. 在模板结尾增加“发布目标平台”约束：
`小红书风格 / 朋友圈截图风格 / 9:16 竖版长图滚动感`。

2. 在模板中加入“禁止项”：
`禁止文字扭曲、禁止居中、禁止低对比度、禁止拼写错误`。
